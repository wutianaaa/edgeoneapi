# 功能完善建议 - 优先级排序

> 已实现功能（速率限制、使用统计、请求日志、熔断器、健康监控页、Anthropic/Gemini 适配器、上游重试、单元测试）已从本清单移除，详见 [README.md](./README.md)。本文件仅保留**尚未实现**的建议。

## ⚡ 中优先级（建议实现）

### 1. 告警通知 🔔
**价值**: 及时发现问题

**功能点**:
- 熔断器打开时发送通知
- 成功率低于阈值告警
- 所有渠道都故障告警
- Webhook 集成（钉钉、飞书、Slack）
- 告警规则配置

**实现难度**: ⭐⭐⭐ (中等)

---

### 2. API Key 管理增强 🔐
**价值**: 更安全的密钥管理

**功能点**:
- API Key 过期时间
- 只读 Key（仅查询统计）
- IP 白名单
- 最后使用时间显示
- 一键轮换 Key

**实现难度**: ⭐⭐ (简单)

> 注：按用户的请求次数统计已随使用统计功能实现。

---

### 3. 使用统计可视化 📈
**价值**: 更直观地了解用量与成本

**功能点**:
- 管理后台图表展示（按用户/渠道/日期）
- 成本估算（基于上游定价）
- 导出为 CSV/Excel

**实现难度**: ⭐⭐⭐ (中等)

> 注：底层 Token 用量记录与查询接口已实现（`/api/admin/usage`），此项为前端可视化补充。

---

## 🎨 低优先级（锦上添花）

### 4. 请求缓存机制 ⚡
**价值**: 减少重复请求，降低成本

**功能点**:
- 使用 EdgeOne KV 缓存相同请求
- 仅缓存确定性请求（temperature=0）
- 可配置缓存时间与失效策略
- 缓存命中率统计

**实现难度**: ⭐⭐⭐ (中等)
**注意**: 需谨慎使用，避免返回过时内容

---

### 5. 更多上游支持 🌐
**价值**: 支持更多 AI 服务

**功能点**（已支持 OpenAI、Anthropic、Gemini，待补充）:
- Azure OpenAI
- 本地模型（Ollama）
- 通义千问
- 文心一言

**实现难度**: ⭐⭐⭐ (中等，每个 1-2 小时)

---

### 6. 高级聊天功能 💬
**价值**: 提升用户体验

**功能点**:
- 对话分享（生成公开链接）
- 对话模板
- 对话搜索
- 对话标签分类

**实现难度**: ⭐⭐⭐ (中等)

---

### 7. 国际化（i18n） 🌍
**价值**: 支持多语言

**功能点**:
- 中文/英文切换

**实现难度**: ⭐⭐⭐ (中等)

### 8. 多租户支持 🏢
**价值**: 支持 SaaS 模式

**功能点**:
- 租户隔离

**实现难度**: ⭐⭐⭐⭐ (复杂)

## 🎯 推荐实施顺序

1. **告警通知** - 及时发现问题
2. **API Key 增强** - 安全提升
3. **使用统计可视化** - 用量洞察
4. **请求缓存** - 性能与成本优化
5. **更多上游 / 高级聊天 / i18n / 多租户** - 按需扩展
